📖 How to Use Page Blocks

Welcome to the Page Editor! Below you will find a simple guide on how to add different types of content blocks to your page. Each block serves a specific purpose. Just copy the example, replace the text/images, and paste it into the content field of the page editor.

🏆 1. Hero Section (Main Header) -- NOT AVAILABLE!

{
    "type": "hero",
    "title": "Welcome!",
    "content": "This is an introduction section for your page.",
    "highlight": "Important text",
    "hidden_text": "Extra information that appears when clicking the 'more' button.",
    "button_text": "more"
}

📝 2. Quote

{
    "type": "quote",
    "content": "Your inspirational quote goes here."
}

📄 3. Text Block

{
    "type": "text_block",
    "content": "This is a simple text block where you can write anything."
}

🖼️ 4. Image with Text (Left)

{
    "type": "text_image_left",
    "image_url": "https://example.com/image.jpg",
    "alt_text": "Description of the image",
    "content": "This text describes the image on the left."
}

🔲 5. Image-Text Grid -- NOT AVAILABLE!

{
    "type": "image_text_grid",
    "items": [
        {
            "image_url": "https://example.com/image1.jpg",
            "alt_text": "Image 1",
            "text": "Description of Image 1"
        },
        {
            "image_url": "https://example.com/image2.jpg",
            "alt_text": "Image 2",
            "text": "Description of Image 2"
        }
    ]
}

📊 6. Table

{
    "type": "table",
    "headers": ["Column 1", "Column 2"],
    "rows": [
        ["Row 1, Column 1", "Row 1, Column 2"],
        ["Row 2, Column 1", "Row 2, Column 2"]
    ]
}

🌆 7. Full-Width Image

{
    "type": "full_image",
    "image_url": "https://example.com/big-image.jpg",
    "alt_text": "Big image description"
}

✅ 8. Lists

{
    "type": "lists",
    "unordered_list": [
        "Bullet point 1",
        "Bullet point 2"
    ],
    "ordered_list": [
        "Step 1",
        "Step 2"
    ]
}

🎬 9. Media (Video and Audio)

{
    "type": "media",
    "video_url": "https://example.com/video.mp4",
    "audio_url": "https://example.com/audio.mp3"
}

🔗 Direct Links to Dropbox Files

If you are hosting mediafiles (video or audio) on Dropbox, you can generate direct links by modifying the shared URL. Replace the ending ?dl=0 with ?raw=1 to get a direct file link.

Example:

Original: https://www.dropbox.com/s/yourfile?dl=0
Direct Link: https://www.dropbox.com/s/yourfile?raw=1

✅ How to Add Blocks to a Page?

Insert the blocks inside []

  1. Open the Pages tab in the admin panel.
  2. Click Add Page or edit an existing one.
  3. Copy a block example from this guide.
  4. Paste it into the Content field.
  5. Modify the text, links, or images.
  6. Click Save to update your page.

That’s it! Your page is now updated 🎉
